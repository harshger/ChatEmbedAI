# A/B Test Setup Skill

## Testing Framework
1. Identify what to test (hypothesis)
2. Define success metric
3. Calculate required sample size
4. Create variations (A and B only)
5. Run test until significant
6. Analyze and implement winner

## What to Test (Priority)
1. Headlines and value propositions
2. CTA text and button color
3. Pricing page layout
4. Form fields and length
5. Social proof placement

## Statistical Requirements
- Minimum 95% confidence level
- Sufficient sample size (use calculator)
- Run for full business cycles (min 1-2 weeks)
- One variable per test
- Document hypothesis before starting
