## German Market Override
DSGVO: Analytics nur nach Cookie-Einwilligung
GA4: Anonymisierung aktivieren (Pflicht in DE)
Alternative: Matomo (deutsch, DSGVO-nativ)
IP-Anonymisierung immer aktiv
Keine US-Server ohne SCC-Vereinbarung
Datenschutzerklaerung: Analytics erwaehnen

# Analytics Tracking Skill

## GDPR-Compliant Setup
1. Cookie consent banner before any tracking
2. IP anonymization enabled
3. Data processing agreement with provider
4. Privacy policy updated with tracking info
5. User opt-out mechanism

## Recommended Tools for Germany
1. Matomo (self-hosted, GDPR-native)
2. Google Analytics 4 (with consent + anonymization)
3. Plausible Analytics (EU-hosted, no cookies)
4. Fathom Analytics (privacy-focused)

## Key Events to Track
- Page views and sessions
- Form submissions
- Button clicks (CTAs)
- Scroll depth
- Conversion goals (signup, purchase)

## Implementation Checklist
- Cookie consent integration
- Event tracking setup
- Conversion goals defined
- Dashboard configured
- Regular reporting schedule
