# Churn Prevention Skill

## Early Warning Signals
- Decreasing login frequency
- Reduced feature usage
- Support tickets increasing
- Payment failures
- No engagement with emails
- Not using key features

## Retention Strategies
1. Proactive check-in emails at risk indicators
2. Personalized re-engagement campaigns
3. Feature adoption nudges
4. Customer success outreach
5. Win-back offers for churned customers

## Cancellation Flow Best Practices
- Ask for cancellation reason (required)
- Offer alternatives (pause, downgrade)
- Show what they'll lose
- Offer retention discount (last resort)
- Make it easy to cancel (legal requirement in EU)
- Send confirmation and feedback request

## German-Specific Retention
- Transparent cancellation process (Kündigungsbutton pflicht)
- No dark patterns (illegal under German consumer law)
- Offer SEPA Lastschrift to reduce payment failures
- Personal support in German language
- Regular product updates to show value
