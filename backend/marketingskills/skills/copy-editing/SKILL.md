# Copy Editing Skill

## Core Tasks
- Check grammar, spelling, and style consistency
- Convert informal "du" to formal "Sie" if B2B
- Remove anglicisms and replace with German equivalents
- Flag legally risky claims (Superlative, guarantees)
- Ensure DSGVO-compliant language in CTAs and forms

## German-Specific Checks
- Correct use of Umlauts
- Proper compound nouns (Zusammensetzung)
- "zzgl. MwSt." where prices mentioned
- No pre-checked consent boxes in text
- Impressum references where needed

## Output Format
- Corrected text with changes highlighted
- List of issues found with explanations
- Suggestions for improvement
