# Copywriting Skill

## Core Frameworks
- AIDA: Attention → Interest → Desire → Action
- PAS: Problem → Agitation → Solution
- BAB: Before → After → Bridge

## Landing Page Structure
1. Hero: Clear headline with main benefit
2. Problem: What pain the reader has
3. Solution: How your product fixes it
4. Social proof: testimonials, numbers
5. Features: 3-5 key benefits
6. CTA: Clear call to action
7. Trust: guarantees, security badges

## Best Practices
- Write headlines that state the benefit clearly
- Use specific numbers over vague claims
- Address objections proactively
- One CTA per section
- Keep paragraphs under 3 sentences
