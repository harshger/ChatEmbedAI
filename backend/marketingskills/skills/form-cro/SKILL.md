# Form CRO Skill

## Optimization Checklist
1. Minimize required fields
2. Use inline validation
3. Clear field labels (not just placeholders)
4. Logical field order
5. Mobile-friendly layout
6. Error messages next to fields

## GDPR Form Requirements
- Consent checkbox (unchecked by default)
- Link to privacy policy
- Clear purpose statement
- Data retention info
- Right to withdraw

## Conversion Boosters
- Trust badges near submit button
- Social proof (number of signups)
- Clear value proposition above form
- Auto-save for long forms
- Confirmation page with next steps
