# Free Tool Strategy Skill

## Strategy Framework
1. Identify audience pain point
2. Build simple tool that solves it
3. Make it free, no registration required
4. Capture leads optionally after value delivery
5. SEO-optimize the tool page

## Popular Tool Types
- Calculators (ROI, pricing, savings)
- Checkers (SEO, GDPR, accessibility)
- Generators (email subject, headline)
- Graders (website, marketing, security)
- Templates (downloadable resources)

## Conversion Strategy
- Tool provides instant value
- Optional email for detailed report
- Upsell to paid product naturally
- Share results on social media option
