# Launch Strategy Skill

## Pre-Launch Phase (4-8 weeks before)
1. Build waitlist / landing page
2. Create teaser content on social media
3. Reach out to beta testers
4. Prepare press kit and media list
5. Set up tracking and analytics
6. Write launch copy (emails, social, blog)

## Launch Day Checklist
- Send announcement email to all subscribers
- Post on all social channels
- Submit to Product Hunt / AppSumo
- Send press releases to media contacts
- Engage with every comment and question
- Monitor analytics in real-time
- Fix any critical bugs immediately

## Post-Launch Phase (1-4 weeks after)
- Collect and respond to feedback
- Write case studies from early users
- Optimize based on real usage data
- Follow up with media contacts
- Run retargeting campaigns
- Plan next feature release

## German Market Launch Tips
- Focus on quality and reliability messaging
- Emphasize DSGVO compliance and data security
- Use German-language press releases
- Target industry-specific publications
- Consider IHK partnerships for credibility
