# Lead Magnets Skill

## Popular Formats in Germany
1. PDF Checklists (most popular)
2. Guides and Leitfaeden
3. Templates and Vorlagen
4. Calculators and tools
5. Whitepapers (B2B)

## Creation Framework
1. Identify target audience pain point
2. Choose format that solves it fastest
3. Create high-value content
4. Design professional PDF
5. Write compelling landing page copy
6. Set up GDPR-compliant opt-in form

## GDPR Requirements
- Clear consent for email marketing
- Double Opt-In after download
- Privacy policy link on form
- No bundled consent
- Data retention statement
