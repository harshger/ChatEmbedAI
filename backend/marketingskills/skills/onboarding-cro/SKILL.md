# Onboarding CRO Skill

## Onboarding Framework
1. Welcome screen with clear first action
2. Progressive disclosure (don't overwhelm)
3. Quick win within first session
4. Checklist for key activation steps
5. Follow-up email sequence

## Key Metrics
- Time to First Value (TTV)
- Activation rate (completed key action)
- Day 1/7/30 retention
- Feature adoption rate

## Best Practices
- Show progress (checklist, progress bar)
- Celebrate milestones
- Offer help proactively
- Personalize based on use case
- Send targeted reminders for incomplete steps
