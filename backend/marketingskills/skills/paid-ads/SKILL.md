## German Market Override
Google.de dominiert in Deutschland (94%)
Meta: DSGVO-konforme Custom Audiences
Kein Targeting auf sensible Kategorien
Anzeigentexte: sachlich, keine Superlative
Pflicht: korrekte Preisangaben mit MwSt.
Abmahnung-Risiko: keine "guenstigste" oder "beste"
Retargeting: Cookie-Einwilligung pruefen
LinkedIn/Xing fuer B2B ueber 50 EUR CPC rentabel

# Paid Ads Skill

## Campaign Strategy
1. Define goals (leads, signups, sales)
2. Set budget and target CPA
3. Choose platforms based on audience
4. Create ad groups and targeting
5. Write ad copy (German market rules)
6. Set up conversion tracking

## Platform Priority for Germany
1. Google Ads (94% search market share)
2. LinkedIn (B2B, higher CPC but quality)
3. Meta/Instagram (B2C, visual products)
4. Xing (B2B German market)

## Budget Planning
- Start with search campaigns
- Test with minimum viable budget
- Scale winners, kill losers weekly
- Reserve 20% for testing new approaches
