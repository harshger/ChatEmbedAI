# Paywall Upgrade CRO Skill

## Upgrade Page Framework
1. Show value before price
2. Feature comparison table
3. Social proof from upgraders
4. Clear pricing with VAT info
5. Money-back guarantee
6. Easy cancellation info

## Pricing Psychology
- Anchor with highest plan first
- Highlight recommended plan
- Show annual savings
- Use "zzgl. MwSt." for B2B
- SEPA as payment option

## Trust Elements for German Market
- 14-day money-back guarantee (Widerrufsrecht)
- Cancel anytime, no minimum term
- DSGVO-compliant data handling
- EU server location
- German customer support
