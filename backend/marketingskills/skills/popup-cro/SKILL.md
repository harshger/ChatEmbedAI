# Popup CRO Skill

## Popup Types
1. Exit-intent: triggers when mouse moves to close tab
2. Scroll-based: after 50-70% page scroll
3. Time-delayed: 30-60 seconds minimum
4. Click-triggered: on specific CTA elements

## Best Practices
- Clear close button (X) always visible
- One clear value proposition
- Single CTA per popup
- Mobile: max 50% screen height
- Frequency cap: once per session

## GDPR Compliance
- No pre-checked consent boxes
- Clear opt-in language
- Link to privacy policy
- Easy to dismiss
- Respect cookie consent settings
