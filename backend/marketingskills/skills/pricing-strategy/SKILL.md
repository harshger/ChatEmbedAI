# Pricing Strategy Skill

## Pricing Frameworks
- Value-based pricing: price based on value delivered
- Tiered pricing: 3-4 tiers for different segments
- Usage-based: charge per unit of consumption
- Freemium: free tier to drive adoption

## Package Design
1. Name plans clearly (not creative names)
2. Show 3-4 tiers maximum
3. Highlight recommended plan visually
4. List features in order of importance
5. Show annual savings prominently

## Psychological Pricing
- Anchor with highest price first
- Use charm pricing (€29 not €30)
- Bundle features strategically
- Create clear upgrade paths between tiers
