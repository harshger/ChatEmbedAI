# Referral Program Skill

## Program Design Framework
1. Define reward structure (two-sided recommended)
2. Set up tracking mechanism
3. Create sharing tools (links, emails, social)
4. Design referral landing page
5. Build automated reward fulfillment

## Reward Models
- Two-sided: Both referrer and referee get value
- Credits: Account credit for successful referrals
- Discount: Percentage off for both parties
- Feature unlock: Premium features for referrals
- Cash: Direct payment (highest motivation)

## Promotion Strategy
- In-app prompts after positive moments
- Email campaigns to satisfied customers
- Dedicated referral page in dashboard
- Social sharing with pre-written copy
- NPS survey → redirect to referral for promoters

## German Market Considerations
- Clear terms and conditions (AGB für Empfehlungsprogramm)
- DSGVO-compliant data handling for referred contacts
- Transparent reward delivery timeline
- Tax implications note for cash rewards
- Written consent before contacting referred person
