# Signup Flow CRO Skill

## Optimization Framework
1. Reduce friction (fewer fields)
2. Build trust (security badges, DSGVO)
3. Clear value proposition above form
4. Social proof near CTA
5. Error handling in German

## German Legal Requirements
- Double Opt-In for email marketing (mandatory)
- No pre-checked marketing consent boxes
- Privacy policy link next to submit button
- Clear data usage explanation
- Right to withdraw consent

## Best Practices
- Progressive profiling over multiple steps
- Only required fields: email, password
- Social login options (Google, Xing)
- Mobile-optimized form layout
- Success confirmation with next steps
