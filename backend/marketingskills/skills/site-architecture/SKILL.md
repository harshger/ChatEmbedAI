# Site Architecture Skill

## Website Structure Planning
1. Define page hierarchy (max 3 levels deep)
2. Plan URL structure (clean, no umlauts)
3. Create navigation menu structure
4. Plan internal linking strategy
5. Define required legal pages

## German Legal Pages (Required)
- /impressum (legal notice)
- /datenschutz (privacy policy)
- /agb (terms and conditions)
- /kontakt (contact page)
- /widerruf (cancellation policy, if applicable)

## SEO Architecture
- Flat hierarchy (important pages max 2 clicks from home)
- Topic clusters with pillar pages
- Breadcrumb navigation
- XML sitemap with hreflang for DE/AT/CH
- Canonical tags for duplicate content prevention
